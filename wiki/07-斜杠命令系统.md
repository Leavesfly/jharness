# 07 - 斜杠命令系统

> 位于 `io.leavesfly.jharness.command`。在交互式模式下以 `/` 开头的指令，由 `CommandRegistry` 统一分发。

## 1. 核心接口

```java
public interface SlashCommand {
    String getName();
    String getDescription();
    CompletableFuture<CommandResult> execute(String[] args,
                                             CommandContext ctx,
                                             Executor executor);
}
```

简化实现 `SimpleSlashCommand` 直接接受 lambda：

```java
new SimpleSlashCommand("status", "会话状态", (args, ctx, exec) -> {
    // 返回 CommandResult.success("...") / .failure("...")
});
```

### `CommandContext` 字段

```java
Settings settings;
QueryEngine engine;
ToolRegistry toolRegistry;
PermissionChecker permissionChecker;
SkillRegistry skillRegistry;
...
```

### `CommandResult` 结构

```java
{ boolean success, String message, Map<String,Object> data }
```

## 2. `CommandRegistry`

### 2.1 两种构建方式

```java
// 方式一：仅注册内置轻量命令
new CommandRegistry();

// 方式二：注入全量依赖，启用 MCP / Agent / Cron / Hooks 命令
CommandRegistry.createFullRegistry(settings, engine, toolRegistry,
    mcpManager, teamRegistry, orchestrator, cronRegistry, hookRegistry);
```

### 2.2 查找协议

```java
registry.lookup("/help")
registry.lookup("/status 详细")   // 自动裁剪空格后的 args
registry.lookup("help")            // 省略 /
```

内部转 lowercase，允许忽略大小写。

## 4. 命令执行流

```
用户输入 "/status 详细"
    │
    ▼
ConsoleInteractiveSession / TerminalUI
    │
    ▼
CommandRegistry.lookup("/status")  ── 返回 SlashCommand
    │
    ▼
cmd.execute(["详细"], context, executor)
    │
    ▼
CommandResult.success("Messages: 8\nUsage: input=...")
    │
    ▼
UI 渲染输出
```

**特殊处理**：
- `exit` / `clear` 返回 success 但携带特定 message，由 UI 层识别以触发真正的退出 / 清屏。
- 长耗时命令（如 `/resume` 加载大文件）返回 `CompletableFuture`，UI 显示 loading。

## 5. 快捷键系统（`command/keybindings/`）

| 类 | 职责 |
|----|------|
| `DefaultKeybindings` | 内置默认映射（`Ctrl+C` 取消、`Ctrl+D` 退出等） |
| `KeybindingParser` | 解析字符串如 `"Ctrl+Shift+R"` |
| `KeybindingLoader` | 从 settings 或文件加载 |
| `KeybindingResolver` | 运行时根据 `KeyStroke` 查找对应行为 |

默认绑定（摘录）：
- `Ctrl+C` → 取消当前查询（`QueryEngine.cancel()`）
- `Ctrl+D` → 退出
- `Ctrl+L` → 清屏
- `ArrowUp / ArrowDown` → 历史输入浏览

用户可通过 `/keybindings` 查看和配置。

## 6. 自定义命令

```java
SlashCommand myCmd = new SimpleSlashCommand(
    "mycmd", "我的命令",
    (args, ctx, exec) -> {
        String response = "参数：" + String.join(", ", args);
        return CompletableFuture.completedFuture(CommandResult.success(response));
    });
registry.register(myCmd);
```

也可通过 **Plugin**（见 [10](10-插件与技能系统.md)）提供命令扩展。
